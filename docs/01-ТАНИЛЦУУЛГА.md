# СДЗН Платформ - Танилцуулга

## Тойм

СДЗН (Социал Демократ Залуучуудын Нэгдэл) платформ нь гишүүнчлэлийн удирдлагын цогц систем юм. Энэхүү систем нь байгууллагын гишүүдийг бүртгэх, удирдах, арга хэмжээ зохион байгуулах, гишүүнчлэлийн татвар хураах зэрэг үйл ажиллагааг автоматжуулах зорилготой.

## Үндсэн боломжууд

### 1. Гишүүнчлэлийн удирдлага
- Гишүүн бүртгэх, засварлах, устгах
- Гишүүний түүх хадгалах
- Гишүүний статус удирдах (идэвхтэй, түр зогсоосон, цуцалсан)
- Гишүүний хайлт, шүүлтүүр

### 2. Байгууллагын бүтэц
- Үндэсний түвшин
- Аймаг/Хотын түвшин
- Дүүрэг/Сумын түвшин
- Шатлал дагасан эрхийн удирдлага

### 3. Арга хэмжээний удирдлага
- Арга хэмжээ үүсгэх, засварлах
- Бүртгэл хүлээн авах
- Ирц бүртгэх
- Оролцогчдын тайлан

### 4. Гишүүнчлэлийн татвар
- Татвар үүсгэх
- Төлбөр хүлээн авах
- Өрийн удирдлага
- Санхүүгийн тайлан

### 5. Тайлан, статистик
- Гишүүнчлэлийн тайлан
- Санхүүгийн тайлан
- Арга хэмжээний тайлан
- Dashboard
